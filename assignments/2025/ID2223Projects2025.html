ID2223 Project Showcase 2025

This page showcases some of the Serverless ML Systems developed by the students in the Scalable ML and Deep Learning masters level course (ID2223 at KTH university). The main requirements for the project were to build a complete ML system that includes:

  • a novel data source that is periodically updated
  • a prediction problem to solve for that data source
  • a user interface or dashboard to communicate the value of your ML system’s predictions

In practice, these projects followed the feature-training-inference pipeline architecture for building ML systems. The programs developed included most of the following:

  • a feature pipeline to produce features from a data source that is continually updated and write the features to a feature store,
  • a training pipeline that creates a consistent snapshot of feature data from the feature store, although a few projects used a pretrained model and didn’t include a training pipeline,
  • an inference pipeline to make predictions on new data and a UI for the prediction problem.

Nearly all projects run on free serverless ML infrastructure in the cloud, although a couple of projects with custom UIs used nodejs. Most of the projects used some variant of the following free serverless services such as Hopsworks, Modal, Github Actions, and Hugging Face (Gradio/Streamlit). A couple of them fine-tune LLMs.
